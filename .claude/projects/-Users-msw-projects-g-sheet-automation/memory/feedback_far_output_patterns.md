FAR output should match the pattern seen in KFA office sheets:
- PDFs and lookup tables = baseline reference / quick lookup
- Screenshots = real-world office output pattern to match
- Site-specific ordinances / linked PDFs = higher-priority override sources when present

Specifically:
1. Always show allowable FAR with citation (e.g., "4.0:1 PER SAN PEDRO CPIO SECTION III-2.B.4")
2. When entitlement modifies FAR, show both the base and the entitlement (e.g., "4.11 FAR BASE" per DIR)
3. Proposed FAR = proposed floor area / lot area, shown as separate calculation
4. Per-floor FAR area breakdown table in the output
5. Definitions section citing LAMC 12.03 or applicable floor area definition

**Why:** The screenshots show the exact format architects expect. Matching this reduces friction in adoption.
**How to apply:** When generating FAR output for G-sheet, follow the layout pattern from these reference sheets.
